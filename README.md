- Sources : https://mermaid.js.org/config/usage.html

## Installation

```bash
pip install flask
pip install flask_socketio
```

Run the program using 
```bash
python3 -m venv venv
source venv/bin/activate
#apres avoir instaler tout on pourra executer

python3 app.py 
```
